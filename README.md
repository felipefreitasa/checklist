## Technologies

React Native, Expo, Typescript and Styled-components


## Functionalities

- Create a to-do
- Delete a to-do
- Update a to-do status (checked/unchecked)




## Running locally

Clone the project

```bash
  git clone https://github.com/felipefreitasa/checklist.git
```

Enter the project directory

```bash
  cd checklist
```

Install the dependencies

```bash
  yarn
```

Start the app

```bash
  yarn start
```
